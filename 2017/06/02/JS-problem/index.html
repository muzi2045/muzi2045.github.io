关于JS里面的强制取整问题

由于Javascript采用的是浮点数值计算, 常常会出现对一个数值进行强制取整的情况,

现总结如下:

  1. 丢弃小数部分,保留整数部分

    js: parseInt(7/2) = 3

  2. 向上取整, 有小数就整数部分加1

    js: Math.ceil(7/2) = 4

  3. 四舍五入

    js: Math.round(7/2)

  4. 向下取整

    js: Math.floor(7/2)
