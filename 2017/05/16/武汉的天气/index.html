武汉的天气

发表于 2017-05-16 |

​ 刚过完五一, 现在正处在武汉的梅雨季节, 天气时热时冷, 今天是学校实训放假的最后一天.

明天也就是星期三又要去机房培训, 不知道那个培训机构的老师又要讲前端的什么东西, 前段时间让我们抄了一个百度的首页, 吐血QAQ…….

我还是更感兴趣在于后端的一些逻辑包括技术栈的学习和扩充.

​ 目前在python方面, 对Django练过几个小页面, Flask还没有接触, 日后一段准备看一下它的源码,毕竟Flask更加精简, 源码大概在5W行左右, 争取能看下来一部分(全部看完希望不大).

​ 在JavaScript这一块刚刚开始起步, 了解了Node.JS, Vue.JS以及各种小的插件, 目前感觉node.js里面的npm或者yarn(都是包管理器,貌似yarn好用一点)等同于python里面的pip, python里面的jinjia2模板渲染引擎等同于node里面的nunjucks….好多相同之处, 但是由于JS自带异步优势,尤其是现在ES6的出炉, async…. await出现直接去除了很多冗余代码, 个人目前比较看好的技术栈还是:JS+Node.JS+ Vue&react +mongDB&mysql, 当然目前还有一些东西需要接触, 比如webpack, Babel, ESLint, Compat, Flow, Husky…接下来一个月慢慢接触
